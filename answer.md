## dict
<p id="dict"></p>

```python{cmd}
# Create a dictionary representing a person
person = {
    'name': 'Alice',
    'age': 30,
    'city': 'New York'
}

# Access specific elements in the dictionary
print("Name:", person['name'])
print("Age:", person['age'])
print("City:", person['city'])
```
