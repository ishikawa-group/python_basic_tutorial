# function
* In this lecture, we learn how to define and use function in python.
* Defining function
```python
def say_hello():
  return "hello"
```
* Usin function
```python
say_hello()
```
* You can pass *argument* to the function, like
```python
def say_someting(s):
  return s

say_something("hello")  # => "hello"
```
